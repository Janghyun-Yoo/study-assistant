# Template References

This project may use public Markdown note systems as structural inspiration.
Do not copy external templates verbatim unless the license permits it and the
source is attributed.

## References Reviewed

### edurbrito/markdown-notes

- URL: https://github.com/edurbrito/markdown-notes
- License: MIT
- Useful patterns:
  - Markdown note templates for study workflows
  - Q&A structure for active recall
  - HTML/table-heavy note blocks
  - Horizontal rules used as PDF page-break markers

### Foam Note Templates

- URL: https://docs.foamnotes.com/features/templates/
- Useful patterns:
  - Markdown templates with metadata/front matter
  - Template variables for title, date, and slug
  - Filepath-aware note creation
  - Preference for simple Markdown templates when portability matters

## Applied to Study Assistant

The compact exam note template uses:

- YAML-like metadata for source, topic, system, and storage paths
- A first-page exam snapshot
- Table-heavy diagnosis/treatment/drug sections
- Active-recall Q&A
