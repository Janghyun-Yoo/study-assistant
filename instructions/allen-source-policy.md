# Allen's Library Source Policy

This project is designed for personal study support while using Allen's Library.

## Allowed Use

- Summarize or restructure material for the user's private study.
- Extract key exam points from a page the user is actively studying.
- Convert the user's own notes and reasoning into review material.
- Create compact notes, checklists, tables, and quiz questions for personal use.

## Avoid

- Copying large sections of Allen's Library text into repository files.
- Creating substitute materials that replicate paid content.
- Publishing generated notes as if they were original official material.
- Removing attribution when a note is based on Allen's Library.

## Recommended Attribution

Generated notes should include:

```text
Source studied: Allen's Library
Page title:
URL:
Generated for personal study:
```

## Practical Rule

Use Allen's Library as the study source, but make the output a transformed,
compressed, exam-oriented personal note.
